kBitArray Class Reference
kApi/Data

Description

Represents a 1D array of bits.

The kBitArray class represents a 1D array of bits. kBitArray is similar in spirit to kArray1<kBool>, but is optimized for space and provides a set of convenient bitwise operations.

kBitArray supports the kObject_Clone, kObject_Dispose, kObject_Equals, and kObject_Size methods.

kBitArray supports the kdat6 serialization protocol.
