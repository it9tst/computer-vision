kProcess Class Reference
kApi/Utils

Description

Represents a process.

kProcess class allows to start a process and offers the possibility to define multiple arguments. For Input and output kPipeStream class can be used. For a kPipeStream of stdin, stdout or stderr for the current process see kStdIn(), kStdOut() and kStdErr() in kPipeStream.h.
